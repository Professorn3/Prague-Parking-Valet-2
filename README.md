Prague Parking 2.1

A C# console application for managing a parking garage using JSON for data storage.

Features

    Park, unpark, and move vehicles (Car, MC, Bus, Bicycle).

    Saves all data to parking_data.json so no data is lost on exit.

    Calculates parking costs automatically.

    Visual map of the parking garage in the console.

How to Run

    Clone this repository.

    Open the .sln file in Visual Studio.

    Set PragueParking.App as the startup project and press F5.

Configuration

All settings (like prices and garage size) are stored in config.json. You can edit this file to change how the garage works.
